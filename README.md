# 3기_도서관대출서비스_김승수

도서관대출서비스 프로젝트 템플릿입니다.
11/16 template 기본구성 구상 및 기본사항일부 작성
로그인, 로그아웃
- session을 통한 관리
회원가입
- 이메일 형식 아이디
- 비밀번호는 영문, 숫자, 특수문자 중 2종류 이상을 조합하여 최소 10자리 이상 또는 3종류 이상을 조합하여 최소 8자리 이상의 길이로 구성
메인페이지
- db(sqlite)내의 모든책 정보
- 현재 남은 책의 수
- 책소개 페이지
- 책 평점
- 한페이지에 8권씩
대여
- 책 대여기능

- 이미 대여한 책의 경우에도 불가
반납
- 현재 대여한 책 목록
- 반납 기능
책소개
- 책 소개
- 댓글 작성 기능
- 평점 입력 기능
대여기록
- 지금까지 대여, 반납한 모든 책에대한 기록 출력
